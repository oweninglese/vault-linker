# Tagfile Format

The tagfile defines the approved vocabulary.

Vault-Linker will only create links for terms in this file.

---

# Example

Canada
Treaty 9
Machine Learning
Artificial Intelligence

---

# Supported Formats

Comma separated

Canada, Treaty 9, Machine Learning

Line separated

Canada
Treaty 9
Machine Learning

---

# Comments

Lines beginning with // are ignored.

Example:

// Geography
Canada
USA

---

# Normalization

The system:

trims whitespace  
deduplicates entries  
preserves original case
