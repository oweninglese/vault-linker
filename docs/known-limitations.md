# Known Limitations

Vault-Linker intentionally avoids certain features.

---

# Lexical Matching

Links are based on literal text matches.

Semantic understanding is not implemented.

---

# Single-pass linking

Currently links only the first occurrence of a term per file.

Future versions will support configurable link ratios.

---

# No real-time updates

Vault-Linker is a batch processor.

It does not watch files continuously.
